# 新三国梗术语库

> 格式：`# 分类 — agent_id` → `## 子类型` → 台词列表（RAG 语料）。
> `→ 外部: xxx` 表示该台词可能调用的外部资源。
> 子类型提供额外上下文，同一 Agent 下的不同子类型可走不同处理分支。
> 后续由 LLM 处理并自动生成 `data/memes.yaml`。
